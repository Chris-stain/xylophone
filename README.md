# What is Xylophone
A iOS 12 project that plays 7 music notes (C D E F G A B)


Each of the 7 notes correspond to a respective color in th UI.

* C (Red)
* D (Orange)
* E (Yellow)
* F (Green)
* G (Turquoise)
* A (Blue)
* B (Purple)

## Frameworks
* UIKit
* AVfoundation
